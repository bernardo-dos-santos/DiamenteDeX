# Diamente De X
![](https://i.imgur.com/e130K9t.gif)
## Introdução 
Bem vindo a um programa, onde, quando você digita um número ímpar, é gerado um diamente de acordo com o número

## Detalhes
Quando o usuário digita um número válido, é gerado um desenho de um diamante com o tamanho ordenado pelo número digitado

## Funcionalidades
**Tratamento de Entrada** - É pedido ao usuário digitar um número ímpar, caso o mesmo não seja feito, é solicitado digitar novamente

**Exibição de Desenho** - É exibido ao usuário um diamante de X's , onde seu tamanho de linhas e colunas máximas é o mesmo do número digitado

**Laço de Repetição** - Caso o usuário deseje, é possível digitar infinitas vezes que você desejar, desde que seja um número ímpar

## Requisitos para uso
.NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto

## Como utilizar
Clone o Repositório
```
git clone https://github.com/bernardo-dos-santos/DiamenteDeX
```
Navegue até a pasta raiz da solução
```
cd DiamenteDeX
```
Restaure as dependências
```
dotnet restore
```
Navegue até a pasta do projeto
```
cd DiamenteDeX.ConsoleApp
```
Execute o projeto
```
dotnet run
```
